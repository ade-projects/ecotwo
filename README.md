# EcoTwo : Proyek Analisis Laju Pertumbuhan Ekonomi Kaltim (2016-2019)

Proyek ini merupakan bagian dari tugas mini project Dasar Pemrograman. Tujuannya adalah untuk mengevaluasi "Laju Pertumbuhan Ekonomi Provinsi Kalimantan Timur selama periode **2016 sampai 2019**".

**Sumber Data**:

- **Dataset**: [Data Laju Pertumbuhan Ekonomi Provinsi Kaltim Tahun 2016-2019](https://data.kaltimprov.go.id/dataset/data-laju-pertumbuhan-ekonomi-provinsi-kaltim-tahun-2016-2019)
- **Penyedia**: BPS Prov. Kaltim
